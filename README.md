Super-simple, minimum abstraction Lever DATA API v3 wrapper in PHP with support for Laravel.

## Installation

You can install the package via composer:

```bash
composer require via-work/lever-php
```

## Usage

``` php
 Lever::api();
```

### Testing

``` bash
composer test
```
